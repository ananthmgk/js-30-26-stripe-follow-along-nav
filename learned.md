## Child Selector (>).
```
div#first > p {
  background: yellow;
}

<div id="first">
  <p>This paragraph will be selected</p>
  <div>
    <p>This paragraph will not be selected</p>
  </div>
</div>
```
`.getBoundingClientRect()` will tell the height, width positions etc, details of the the element.
